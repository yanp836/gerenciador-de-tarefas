# 📝 Gerenciador de Tarefas (Java) 📝

Projeto simples em Java feito no console, usando Programação Orientada a Objetos (POO).

## ⚙️ Funcionalidades ⚙️

- Adicionar tarefas  
- Listar tarefas  
- Concluir tarefas  
- Remover tarefas  

🚀 Como executar  🚀

1. Compile os arquivos:
   ```bash
   javac src/*.java
2. Execute o programa:
   bash
      java -cp src Main

##💻 Tecnologias 💻

   Java
   Programação Orientada a Objetos (POO)
